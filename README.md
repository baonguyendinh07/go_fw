# go_fw
(golang framework)

go_fw is a web framework api written in Go
I build this fw for study.

---------------------------------------------

Several Features:
- Framework resuable core

- Controllers
- Service pattern
- Repository pattern
- Model
- Processing of request data
- Database query builder
- Database ORM

- HTTP request
- Authenticate
- Authorization
- Router
- Middlewares
- Validations
- Response Json

- Migration
- Email
- CLI
- Cache
- Write log
- Catch Exception

(I will update more features if I find out more)
